距离分辨率 ΔR = c / (2B)-
最大不模糊距离 Rmax = fs·c·Tc / (4B)-
Chirp 斜率 S = B / Tc-
距离 bin 宽度 = Rmax / (N/2)-
📐 距离分辨率只取决于带宽 B，与 Chirp 时长无关。B = 4 GHz 时 ΔR ≈ 3.75 cm，这是 77 GHz 车载雷达的典型指标。
⚠ Rmax 受限于 ADC 采样率：拍频 fb ≤ fs/2（奈奎斯特）。
