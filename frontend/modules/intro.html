从「啁啾」开始理解 FMCW

FMCW 雷达发射 频率随时间线性变化 的连续波（Chirp），接收回波后与发射信号做混频，得到频率与目标距离成正比的 拍频信号 (Beat Signal)。这是毫米波雷达（如 TI IWR系列、Infineon BGT60）的核心技术。

📈 线性调频 🎯 距离·速度·角度 ⚡ 低成本低功耗 🚗 车载 · 无人机 · 工业

Chirp 信号波形

调整参数，观察频率 vs. 时间 与 时域波形：

瞬时频率 f(t) = f₀ + S·t
时域信号 cos(2π(f₀t + ½St²))

核心公式速查

Chirp 斜率频率随时间线性增加：

S = B / Tc

发射信号

stx(t) = A · cos( 2π( f0·t + ½·S·t² ) )

回波延迟目标在距离 R 处：

τ = 2R / c

拍频信号混频后得到的中频 (IF) 频率：

fb = S · τ = 2·S·R / c = 2·B·R / (c·Tc)

距离反推

R = fb · c · Tc / (2·B)
💡 关键思想：拍频 fb 与目标距离 R 成 线性正比。对 IF 信号做 FFT，频谱峰值的位置就对应目标的距离。
