④ 角度测量：多接收天线与相位差

当目标不在雷达法线方向时，到达不同 RX 天线的回波存在 路径差，从而产生 相位差。用均匀线阵 (ULA) 即可估计方位角。

ULA 几何

波前   路径差 ΔL = d·sinθ

测角公式

相邻天线相位差

Δφ = 2π · d · sinθ / λ

角度反推

θ = arcsin( Δφ · λ / (2π · d) )

角度分辨率（瑞利限，N 元 ULA）：

Δθ ≈ 0.886 · λ / (N · d · cosθ)

无模糊测角条件

|Δφ| < π   ⟹   d ≤ λ/2   （避免相位模糊）
