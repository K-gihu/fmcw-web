③ 速度测量：Chirp 序列与 2D-FFT

单个 Chirp 只能测距。要测速度，需要发射 一帧 N 个 Chirp，利用相邻 Chirp 之间回波的 相位差 估计多普勒频移。

帧结构参数

目标

TX chirps   RX echoes（多普勒相移逐 Chirp 累积）

多普勒测速公式

载波波长

λ = c / fc

相邻 Chirp 相位差

Δφ = 4π · v · Tc / λ

速度反推对慢时间做 Doppler-FFT 得峰值 fd：

v = λ · fd / 2   其中   fd = Δφ / (2π · Tc)

性能指标

最大不模糊速度 vmax = λ / (4Tc)-
速度分辨率 Δv = λ / (2·Nchirp·Tc)-
帧时长 Tframe = Nchirp·Tc-
🚗 速度分辨率取决于整帧时长。77 GHz 下想达到 0.1 m/s 分辨率，需 Tframe ≈ 19 ms。
